# ESP32 Embedded Web Server

## Project Overview
This project demonstrates an Embedded Web Server using the ESP32 microcontroller.The ESP32 connects to a Wi-Fi network and hosts a web page that allows users to control an LED through a web browser. The project showcases the integration of IoT concepts, wireless communication, and embedded system programming.

## Objectives
- To understand the basics of ESP32 web server implementation.
- To establish Wi-Fi connectivity using ESP32.
- To create a simple web interface for device control.
- To control an LED remotely through a web browser.

## Components Used
- ESP32 Development Board
- LED
- 220Ω Resistor
- Jumper Wires
- Wokwi Simulator

## Software Requirements
- Arduino IDE / Wokwi Simulator
- ESP32 Board Package
- WiFi Library
- WebServer Library

## Working Principle
1. The ESP32 connects to a Wi-Fi network.
2. A web server is created on the ESP32.
3. When a client accesses the server through a browser, a web page is displayed.
4. The web page contains buttons to turn the LED ON and OFF.
5. User actions are processed by the ESP32, which controls the LED accordingly.

## Features
